# CuseHacks-Datathon-Crime-Wave
Completed in 24 hours during the CuseHacks Datathon. Won third place in track 3. It generates a heat map of crime in the Syracuse area.
You must get a mapbox token for this to work (Put it in maptoken.py and make a variable called maptoken = "insert key here"
Install these following programs in your terminal for this to work 
pip install plotly
pip install matplotlib
pip install numpy
pip install pillow
